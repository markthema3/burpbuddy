Build instructions for development or bleeding edge

OSX

  1. brew install maven
  2. Install Java 8
  3. Clone this repo
  4. cd burp
  5. export JAVA_HOME=/Library/Java/JavaVirtualMachines/jdk1.8.0_05.jdk/Contents/Home
  6. mvn package
  7. cd target
  8. copy burpbuddy-0.1-SNAPSHOT.jar where/you/put/burp/stuff
